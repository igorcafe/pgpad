## Building

### Requirements

- A relatively recent build of `npm`
- The Rust toolchain, with a minimum version of 1.85

### Setup

#### 1. Install dependencies

```
npm install
```

#### Build executable

```
npm run tauri build
```

#### To start the dev server

```
npm run tauri dev
```
